Чаты

Тип:

Сначала подключи мост.

Сортировка: по последней активности (сверху — недавние).

Кол-во сообщений показывается при мосте Baileys (./run.sh).

Все чаты с 0 сообщений? Подожди 1–2 мин после подключения и нажми «Обновить список».

Проверить загрузку истории: статистика (JSON) — смотри totalMessages и byChat[].messageCount. В логах моста после подключения должна появиться строка «messaging-history.set». Если totalMessages так и остаётся 0 — WhatsApp часто не отдаёт историю связанным «десктопным» устройствам; новые сообщения будут появляться в реальном времени, старую историю можно подтянуть только по запросу (если в чате уже есть хотя бы одно сообщение).

Анализ через Gemini

Выбери чаты выше и напиши промпт. Будет загружена история сообщений и отправлена в Gemini.

Если в чате только новые сообщения без старой истории — включи эту опцию и нажми «Анализировать»: с телефона запросится ещё порция сообщений (до ~100 за раз). Повтори для большей глубины.
